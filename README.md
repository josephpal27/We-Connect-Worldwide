# We Connect Worldwide — React Website

A modern, responsive business website built for **We Connect Worldwide**, developed using React, Vite, and Tailwind CSS.

Developed independently by me as the **Frontend Developer**, focusing on performance, clean UI, and smooth user experience.

---

## Features

* Fully responsive design
* Built with Tailwind CSS utility-first styling
* Component-based React architecture
* Smooth animations using AOS
* Optimized images and performance-focused UI

---

## Tech Stack

React • Vite • Tailwind CSS • JavaScript • AOS
